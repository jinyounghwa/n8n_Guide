1. **손으로 따라 만들기**
   - 각 예제는 반드시 직접 n8n에서 구현
   - 읽기만 하지 말고 실행해보기

2. **문제 해결하기**
   - 에러 메시지를 읽고 이해하기
   - 디버그 메시지로 데이터 흐름 확인
   - 각 노드의 입출력 검사하기

3. **발전시키기**
   - 기본 예제에서 시작
   - 점진적으로 복잡한 기능 추가
   - 자신의 아이디어 적용하기

4. **정리하기**
   - 배운 내용을 노트에 정리
   - 스스로 설명해보기
   - 새로운 방법 기록하기

# 3장: n8n UI와 워크스페이스

> n8n의 인터페이스와 주요 기능들을 이해합니다.
> 예상 학습 시간: 40분

## n8n 대시보드 구조

### 좌측 사이드바

```
┌─ Dashboard (홈)
├─ Workflows (워크플로우 목록)
├─ Credentials (인증 정보)
├─ Executions (실행 기록)
├─ Settings (설정)
└─ Help (도움말)
```

## 주요 화면들

### 1. 워크플로우 에디터

워크플로우를 만드는 중앙 캔버스입니다.

**구성요소:**
- 캔버스: 노드를 배치하는 공간
- 좌측 노드 팔레트: 사용 가능한 노드 목록
- 우측 패널: 선택된 노드의 설정
- 상단 도구바: 저장, 실행, 활성화 버튼

### 2. 크레덴셜 관리

API 키와 인증 정보를 중앙에서 관리합니다.

**추가 방법:**
1. Credentials 클릭
2. "Add new" 버튼
3. 서비스 선택 (Gmail, Slack, etc.)
4. 인증 정보 입력

### 3. 실행 기록

워크플로우 실행 히스토리를 확인합니다.

**확인 항목:**
- 실행 시간
- 성공/실패
- 입력/출력 데이터
- 소요 시간

## 워크플로우 편집 팁

### 노드 추가

1. 캔버스에 더블클릭
2. 노드 검색
3. 원하는 노드 선택

### 노드 연결

1. 노드의 우측 원형 포트 클릭
2. 다음 노드의 좌측 포트로 드래그
3. 자동으로 연결됨

### 노드 삭제

1. 노드 선택
2. Delete 키 또는 우클릭 → Delete

## 표현식 에디터

### 표현식 입력

텍스트 필드에서 "="을 입력하면 표현식 에디터 열림

```
예: {{ $json.field_name }}
```

### 자동완성

- Ctrl+Space: 자동완성 팝업
- 변수 목록 표시

### 디버깅

- 표현식이 유효한지 실시간 검증
- 오류 메시지 표시

# 9장: 크레덴셜 및 보안

> API 키와 인증 정보를 안전하게 관리합니다.
> 예상 학습 시간: 1시간

## 크레덴셜 관리

### Credential 추가

```
1. 좌측 메뉴 → Credentials
2. "+ Add new" 클릭
3. 서비스 선택 (Gmail, Slack, etc.)
4. 인증 정보 입력
5. 저장
```

### 환경 변수 사용

```
.env 파일에 저장:
CLAUDE_API_KEY=sk-ant-...
SLACK_WEBHOOK=https://hooks.slack.com/...

n8n에서 접근:
{{ $env.CLAUDE_API_KEY }}
{{ $env.SLACK_WEBHOOK }}
```

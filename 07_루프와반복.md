# 7장: 루프와 반복 처리

> 배열을 반복 처리하는 방법을 배웁니다.
> 예상 학습 시간: 1시간

## Loop 노드

### 기본 사용법

```
배열의 각 요소를 하나씩 처리

입력:
[item1, item2, item3, ...]

Loop 내부에서:
각 item마다 실행
{{ $item.json }}으로 접근

출력:
[처리된_item1, 처리된_item2, ...]
```

### 실제 구현

```
설정:
- Mode: Array
- Array: {{ $json.items }}

Loop 내부:
{{ $item.json.name }}
{{ $item.json.price }}
{{ $index }}  (인덱스)
```

## Split In Batches

### 대량 데이터 처리

```
큰 배열을 작은 묶음으로 나누기

입력:
[1, 2, 3, ..., 1000]

Split In Batches (size: 100):
→ 100개씩 10번 반복

효과:
API 속도 제한 우회
메모리 효율성 향상
```

### 실제 예

```
1000개 주문 처리:

[1000개 주문]
  ↓
[Split: 100개씩]
  ↓
[각 배치마다 처리]
  - 배치1 (1-100): 처리
  - 배치2 (101-200): 처리
  - ...
  ↓
[완료]
```

## 중첩 루프

### 복잡한 반복 처리

```
[외부 Loop: 카테고리]
  ↓
  [내부 Loop: 카테고리별 상품]
    ↓
    [처리]
    ↓
  [내부 완료]
  ↓
[외부 완료]
```

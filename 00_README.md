## 💡 학습 팁

1. **손으로 따라 만들기**: 각 예제는 반드시 직접 n8n에서 구현해보세요
2. **테스트하기**: 각 워크플로우는 `Execute Workflow` 버튼으로 테스트하세요
3. **디버깅 활용**: 각 노드의 `Inspect` 기능으로 데이터 흐름을 확인하세요
4. **JSON 이해하기**: n8n은 JSON을 많이 다루므로 기본 이해가 필수입니다
5. **문서 참고**: 공식 n8n 문서 https://docs.n8n.io를 자주 확인하세요

## 🔧 필수 사전 지식

- **JavaScript 기본**: 변수, 함수, 배열, 객체
- **JSON**: 구조와 문법 이해
- **REST API**: 기본 개념 (GET, POST, 상태 코드)
- **환경 변수**: .env 파일 이해

다른 내용은 교재에서 설명합니다!

## 🚀 빠른 시작

로컬에서 n8n이 이미 설치되어 있다면:

```bash
# n8n 서버 시작
n8n start

# 브라우저에서 접속
# http://localhost:5678
```

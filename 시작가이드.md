## 💡 학습 팁

### 효과적인 학습
1. **매일 일정 시간 할당**: 주 5-6시간 추천
2. **실습 우선**: 모든 예제를 직접 만들기
3. **순서 지키기**: 앞 장을 완료 후 다음 장으로
4. **메모 작성**: 배운 내용을 정리하기
5. **문제 해결**: 에러가 나면 직접 디버깅하기

### 시간 관리
```
추천 학습 일정: 4-5주

```

3. 표현식 문법 검토
